------------------------------------------------------
	Tuned Theme
------------------------------------------------------

	Tuned Theme is a simple, clean responsive theme for Wordpress. Tuned theme looks good on any devices(mobile, tablet, nootebook, PC...). Supports custom menus, cudtom header, custom background and RTL languages, translation ready. Custom Front Page template supports the following extras: Hero Unit, Carousel Slider, 1 Column and/or 3 Column Grid Boxes. Templates: full width, front page, Infinite Scroll blog(with Masonry)

------------------------------------------------------
	Features
------------------------------------------------------
	
	- Responsive webdesign
	
	- Translation ready
	
	- Custom menu
	
	- Custom header
	
	-Custom background
	
	- RTL language support
	
	- Widgets: Tuned offers 5 widget boxes.
		- Main sidebar                       : This is the Main Sidebar on the right side of the page.
		- Front Page - Total Width Box       : Appears on the Front Page. This is a total width box. Ideal for the Hero Widget.
		- Front Page - Carousel Box          : Appears on the Front Page. This is a total width box ONLY for Carousel Widget
		- Front Page - Full Width Box        : Appears on the Front Page. Ideal for the Text Widget, and for the Category Posts Widget
		- Front Page - 3 Column Grid Box     : Appears on the Front Page. 3 Column grid ONLY for the Category Posts Widget. (Displays other widgets only in one column)
		
	-Page Templates: full width, front page, infinite scroll blog
